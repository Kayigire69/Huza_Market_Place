# Youth Huza — System Workflow Diagrams

Professional workflow documentation for staff training, client demos, expo, and user manuals.

**Rules:** These files document the **implemented** platform only. They do not change application code.

**How to view:** Open any file on GitHub (Mermaid renders automatically) or in VS Code / Cursor with a Mermaid preview. Export to PNG/PDF with any Mermaid exporter or print from the preview.

**Style:** Green · white · gray · top-to-bottom flow · plain language.
